# *DMFAL*: Deep Multi-Fidelity Active Learning for High-Dimensional Outputs

by [Shibo Li](https://imshibo.com), Zheng Wang, [Mike Kirby](https://www.cs.utah.edu/~kirby/) and [Shandian Zhe](https://www.cs.utah.edu/~zhe/)
